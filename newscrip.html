Publish to GitHub

Make this website live using GitHub Pages:

# Initialize repository and push to GitHub

echo "# colla" >> README.md
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/p001x/colla.git
git push -u origin main

# After pushing, enable GitHub Pages:

1. Go to your repository Settings

2. Select "Pages" in the left menu

3. Set source to "Deploy from main branch"

4. Click "Save"
